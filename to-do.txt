Fonctionnalités: 
Gestion trajets (départ - destination) :
- CRUD trajets avec points de repos (arrêts) (Ex. Antananarivo - Ambatolampy - Antsirabe): nom + arrêts (misy sens) + durée moyenne

Gestion des arrêts: 
- CRUD arrêts (Ex. Antananarivo, Majunga, Ambatolampy, ...): nom

Gestion voyages :
- CRUD voyages: tb + chauffeur + trajet + date heure départ

Gestion chauffeurs
- CRUD chauffeurs: extend table personne (nom, contact)
- disponibilité chauffeur: voir si le chauffeur n'est pas attribué à un voyage pour la date donnée

Gestion clients:
- CRUD clients: extend table personne (nom, contact)

Gestion disponibilité taxi brousse:
CRUD taxi brousses avec places: matricule
- catégories: VIP, ... 
- disponibilité taxibrousse: voir si le tb n'est pas attribué à un voyage pour la date donnée

Gestion réservations:
CRUD réservations: client + voyage + date de réservation
=> il faut lister les  trajets dispo avec filtre (date heure ou moment de la journée: matin, midi, soir)
=> On ajoute un point de départ et  arrivée pour une date heure donnée du voyageur 
et on cherche s'il existe un trajet qui passe par le point de depart et arrive du client apres ca on cree la réservation
=> voir les places disponibles 

Detail reservation:
id reservation + tb + numero place

Gestion paiement
CRUD paiement: reservation + date de paiement + montant payé

Frais:
trajet + catégories taxibrousse + date
